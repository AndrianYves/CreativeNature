CSS

Introduction

Introduction

CSS stands for Cascading Style Sheets, it is used to enlighten and decorate your websites.
Depicting on how the content of your HTML should be shown in your websites.
A CSS code can be placed inside a HTML file, but it would only work for that certain HTML container,
by means a CSS should be saved with the .css extension instead in order for it to be properly utilized
by all of your HTML files.

Syntax

Syntax

The syntax of a CSS is combined by a SELECTOR and a DECLARATION.
Where h1 in here would be the SELECTOR and inside the open and closed brackets are the DECLARATIONS.

Example Code

h1 and p are the SELECTORS and inside their brackets are the DECLARATIONS.

h1{
      Color: blue;
      Text-align: center;
}

p{
      Color: Yellow;
       Text-align: left;
      Font-size: 20px;
}

Usage

Usage

There are three different ways on how you can use the Cascading Style Sheet.


1. External Style Sheet
2. Internal Style Sheet
3. Inline Style

External Style Sheet

Upon using this style sheet, your HTML file should have a link that calls your css file.

EXAMPLE

<link rel="stylesheet" type="text/css" href="yourcssfile.css">

Internal Style Sheet

This style sheet is the other way around of External Style Sheet. People use this method when they want a unique style on the preferred HTML file.

EXAMPLE

<style>
h1{
      color: black;
       text-align: center;
}
</style>

Inline Style

In this style, it is used if you want to place a style on one line.

EXAMPLE

<h1 style="color:red">
Heading 1 </h1>

Color

You can also use CSS to color your website. It can be used in texts, backgrounds, borders and etc.


Example Code

You can set colors via RGB values, HSL values, RGBA values or just the color name

Backgrounds

Backgrounds

Just like the last topic, in here you can also set a background for your website using CSS.
It can either be an image or just a touch of your favorite color.


Example Code for Colors

body{
      background-color: black;
}

Example Code for Images

body{
      background-image: url("image.gif");
}

You can also set on how your background would appear.

Example Code

body{
       background-color: black;
       background-repeat: no-repeat;
        background-position: center;
}

Borders

Borders

In CSS you can specify the style, width or the color of your border.
There are various of styles to choose from for example
a dashed border style defined a dashed border.


Example Code for Borders

h1.dashed{
      border-style: dashed;
}

You can also set the width and height of your border.

Example Code for Borders

h1.dashed{
      border-style: dashed;
      border-width: 200px;
      border-height: 500px;
}

This are just few ways on how you can design your border.

Margins

Margins

In CSS you can also specify if you want to give an element a margin.
Values that can be used in a marging are the following.

Auto - if you want to let the browser calculate the margin.
Length - You can specify your margin in px, pt, cm, etc.
% - You can also specify your margin using percentages.

Example Code for Margins 

h1{

    margin-top: 20px;

    margin-bottom: 20px;

    margin-right: 20px;

    margin-left: 20px;

}

This are just few ways on how you can design your margin.

Height and Width

Height and Width

Just like before, in CSS you can control all the looks of your website,
including the height or width of a chosen element.

Example Code for Height and Width 


h1{

    height: 200px;
    width: 200px;
}

You can also set the maximum width or height of an element with the max code.


Example Code for Max Height and Max Width


h1{

    max-height: 200px;
    max-width: 200px;
}
Box Model

Box Model

Box Model is the term used when css editors talks about design and layouts.
It is a box that wraps every HTML element around.
It is a combination of margins, borders, paddings, and the content.

Example Code for Box Model 


h1{

    height: 200px;
    width: 200px;
    border: solid;
    margin: 20px;
}

This are just few ways on how you can design your box model.

Outline

Outline

CSS Outline is a customizable line that is drawn around elements.
CSS Outline has various properties, some of these are:

outline-style
outline-color
outline-width

Example Code for an Outline

    outline-color:red;
}

This are just few ways on how you can design your outline.
