Introduction

Introduction

CSS stands for Cascading Style Sheets, it is used to enlighten and decorate your websites.
Depicting on how the content of your HTML should be shown in your websites.
A CSS code can be placed inside a HTML file, but it would only work for that certain HTML container,
by means a CSS should be saved with the .css extension instead in order for it to be properly utilized
by all of your HTML files.

Syntax

Syntax

The syntax of a CSS is combined by a SELECTOR and a DECLARATION.
Where h1 in here would be the SELECTOR and inside the open and closed brackets are the DECLARATIONS.

Example Code

h1 and p are the SELECTORS and inside their brackets are the DECLARATIONS.

h1{
      Color: blue;
      Text-align: center;
}

p{
      Color: Yellow;
       Text-align: left;
      Font-size: 20px;
}

Usage

Usage

There are three different ways on how you can use the Cascading Style Sheet.


1. External Style Sheet
2. Internal Style Sheet
3. Inline Style

External Style Sheet

Upon using this style sheet, your HTML file should have a link that calls your css file.

EXAMPLE

<link rel="stylesheet" type="text/css" href="yourcssfile.css">

Internal Style Sheet

This style sheet is the other way around of External Style Sheet. People use this method when they want a unique style on the preferred HTML file.

EXAMPLE

<style>
h1{
      color: black;
       text-align: center;
}
</style>

Inline Style

In this style, it is used if you want to place a style on one line.

EXAMPLE

<h1 style="color:red">
Heading 1 </h1>
