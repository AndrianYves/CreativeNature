Topic 2 - HTML Syntax

The main syntax of html are tags which consists of special characters like >, < and /.
All the content of the html is placed on the body element consisting of text, document,
hyperlinks, etc.

Example Code

<!DOCTYPE html>
<html>
<head>
      <title>HTML Syntax</title>
      </head>
      <body>

      <h1>Sample Code</h1>

            <p>This is a Sample Code</p>
      </body>
      </html>
