Topic 8 - HTML Colors

You can use HTML colors to attract audience and leave a trademark.
For example, Facebook use the word f and the color blue for their logo
and just by seeing that logo people will realize that it's facebook. It can
be used in text,background, and etc.

You can set HTML colors using color names(complicated one), RGB values,
HEX values, HSL values, RGBA values, and HSLA values.

Example Code

      <h1 style="color:Green;">Sample</h1>
