HTML

Introduction

Introduction

HTML stands for Hyper Text Markup Language, it is the skeleton of your website.
It is primarily used for designing purposes in web pages and web application.
HTML can be very forgiving in the making of the structure. An HTML file should
always have <html> and an end tag of </html> to avoid problems and it should have
an extension of .html.

Syntax

Syntax

The main syntax of html are tags which consists of special characters like >, < and /.
All the content of the html is placed on the body element consisting of text, document,
hyperlinks, etc.

Example Code

<!DOCTYPE html>
<html>
<head>
      <title>HTML Syntax</title>
      </head>
      <body>

      <h1>Sample Code</h1>

            <p>This is a Sample Code</p>
      </body>
      </html>

Styles

Styles

Everyone wants their website to be presentable and here comes the styles to make it more appealing.
The syntax for html style is <tagname style="property:value;">

Example Code

This is the format for the background color:
      <body style="background-color:lightblue;">


This is the format for the text color:
      <h2 style="color:red;">Color red</h2>


This is the format for the text size:
      <h1 style="font-size:150%;">Decent size</h1>


This is the format for the text alignment:
      <h1 style="text-align:center;">Centered text</h1>

Text Formatting

Text Formatting

HTML also have formatting elements. Formatting elements like:
<b> for Bold text
<i> for Italic text
<strong> for Important text.
<em> for Emphasized text
and there are many more that you can choose base on your preferences.

Example Code

<b>This text is in bold</b>

            This text is bold.

<i>This text is italic</i>

            This text is italic.

<strong>This text is strong</strong>

            This text is strong and quite similar to bold.

<em>This text is emphasized</em>

            This text is emphasize and quite similar to italic.

Tables

Tables

The <table> tag is used to define an html table.
HTML table consist of:
<tr> tag which is table row.
<th> tag which is table header.
<td> tag which is table data.

Example Code

<table style=width:10%">
      <tr>
            <th>Cars</th>
      </tr>

      <tr>
            <td>Nissan Skyline GTR</td>
      </tr>
</table>

Responsive

Responsive

Having a responsive web design can automatically resize, shrink or enlarge a website
to make it more comfortable for the user wether the user is using a desktop, tablets, and phones.

Example Code
