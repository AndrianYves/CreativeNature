Events

These are the things that happen to HTML elements. JavaScript can react to on these events.

Example of HTML events:

  • HTML input field was changed
  • HTML button was clicked
  • HTML web page has dinished loading

HTML allows event handler attributes with JScript code that will be added to HTML elements.

  • Single Quotes = (element event = 'JavaScript')
  • Double Quotes = (element event = "JavaScript")

HTML Events

  • onchange = HTML element has been changed
  • onclick = User clicks an HTML element
  • onmouseover = User moves the mouse over an HTML element
  • onmouseout = User moves the mouse away from an HTML element
  • onkeydown = User pushes a keyboard key
  • onload = Browser has finished loading the page
