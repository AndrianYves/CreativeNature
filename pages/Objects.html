Definition

Create a JavaScript object with object literal. Object definition can actually span multiple lines.
Example:
var individual = {
lastName: "Doncic",
firstName: "Luka"};

Object Properties

name:value pairs in JavaScript Objects are also known as properties.

  • lastName = Doncic
  • firstName = Luka

Accessing Object Properties

Accessing object properties in two ways:

  • objectName.propertyName
  • objectName["propertyName"]
