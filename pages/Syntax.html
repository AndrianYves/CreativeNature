Syntax

Set of rules on how the JavaScript programs constructed:
Example:

var x, i ,z;
x = 1; i = 9;
z = x + i;

Values

Two types of values:

  • Fixed values or literals
  • Variable values or variables

Literals

Numbers - written with or without decimals.
Exampke: 100, 12.69

Strings - written within single or dobule quotation.
Example: "JavaScript", 'JavaScript'

Variables

Used to store data values. It uses the keyword var for declaring variabels. The equal sign(=) is use for assigning values to variables
Example:
var x;
x = 10;

Operators

JavaScript uses Arithmetic Operators (*-/+) for computing values.
Example:
(1 + 1) * 123

Equal sign (=) is used for assigning values, which is called the assignment operator.
Example: var i;
i = 10;

Expressions

A combination of values, operators, and variables that computes to a value. The computation is called an evaluation.
Example: 1 * 2 evaluates to 2. or x * 2

Keywords

Used to identify actions to be performed. The keyword var will the browser to make a variable.
Example: var i, a;
i = 2 + 1;
a = i * 100;

Comments

Comments statements are not actually executed in Javascript.

Code after the double slashes or // or between the opening comment tag which is /* and the end comment tag */ is considered as a comment.
Example: //Execute the variable

Identifiers

These are names. It is used to name variables. In JavaSCript, the first character mus be a letter, or an underscore (_), or dollar sign ($).

Case Sensitive

All JavaScript identifiers are case sensitive.

The variables firstname and firstName are two different variables.
Example: var firstname, firstName;
firstname = Ahh;
firstName = Ohh;

Camel Case

Hyphens are not allows in JavaScript. The (-) sign is reserved for subtractions.
Example: first-name, last-name, middle-initial

Underscores, Upper Camel Case and Lower Camel Case are allowed in JavaScript
Example:
Underscore: first_name, last_name
Upper Camel Case: FirstName, LastName
Lower Camel Case: firstName, lastName

Character Set

Javascript uses Unicode character set. It cover almost all the characters, symbols, and punctuations in the world.
