Numbers

Numbers can be written with our without decimals.

JavaScript Numbers are always 64-bit Floating Point

Precision

Integers are accurate up to 15 digits.
Example: var x = 9999999999999; // x will be 9999999999999

Number Strings

JavaScript strings can also have numeric content.
Example: var x = 100; //x is number
var y = "100"; //y is string

NaN - Not a Number

NaN or Not a nnumber is reserved word that indicates if a number is not a legal number.
Example: var x = 100 / "Apple"; // x will be NaN

Infinity

This is the value that will return if calculating a number outside the largest possible numbers.
Example: var eg = 1;
while (eg != Infinity){ // Execute until Infinity
eg = eg * eg; }

Hexadecimal

JScript interprets numeric constants as hexadecimal if they are preceeded by 0x.
Example: var x = 0xFF; // x is 255
