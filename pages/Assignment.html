Assignment Operators

Assigns values to JavaScript Variables.

  • "=" = x = y or x = y
  • += = x += y or x = x + y
  • -= = x -= y or x = x - y
  • *= = x *= y or x = x * y
  • /= = x /= y or x = x / y
  • %= = x %= y or x = x % y
  • <<= = x <<= y or x = x << y
  • >>= = x >>= y or x = x >> y
  • >>>= = x >>>= y or x = x >>> y
  • &= = x &= y or x = x & y
  • ^= = x ^= y or x = x ^ y
  • |= = x |= y or x = x | y
  • **= = x **= y or x = x ** y

Example: += adds a value to a variable.

var x = 1;
x += 5;

Example: -= subtracts a value from a variable.

var x = 1;
x -= 5;

Example: *= multiplies a variable.

var x = 1;
x *= 5;
