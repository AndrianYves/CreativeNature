Functions

A block of code that was designed to use for a particular type. It is executed when something calls it.
Example:
function thisFunction(p1){ return p1; //Returns the product p1 }

Function Syntax

Defined with the function keyword, then name, , then parentheses.

It contains letters, underscores, dollar sign ($), and digits.

It includes parameter names that are separated by a comma.

It is placed inside the curly brakets {} for it to be exexuted.
Example:
function name(parameter1, param2...)
{ //code to be executed
}

Return

Return statement will let the function stop executing.

After the invoking statement, JavaScript will return and execute the code. Functions mostly compute a return value, this value is retunred back to the caller.

Local Variables

Variables actually declared within a function, and became LOCAL to the function.

It can accessed from within the function.
