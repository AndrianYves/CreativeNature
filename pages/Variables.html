Variables

The containers for storing data values.

Identifiers

JavaScript variables should be identified with unique names.

These unique names are also known as identifiers.

It can be short names like x and y or more descriptive like name, age, speed, etc.

General Rules for Constructing names for variables:

  • It can contain letters, underscores, dollar signs, and digits.
  • It must begin with a letter.
  • It can also begin with dollar sign ($) and underscore (_)
  • It is case sensitive (variable x and X are different from each other)
  • Reserved words cannot be used as a names.

Assignment Operator

The equal sign (=) is the assignment operator and not an "equal to" operator.
Example: x = i + 1;

The double equal sign (==) is considered as the "equal to" operator.

Data Types

It can hold numbers like 100 and also text values.

Text values are called text strings.

JScript can handle many data types. It is written inside the double or sign quotes. Number are written without using quotes. If you insert a number in quotes, it will be considered as a text string.
Example:
var qw = 123;
var name = "John";
var age = "13";

Declaring Variables

Declare a variable in JavaScript by using the keyword var.
Example: var x;

Assigning a value to the varaible by using the = sign.
x = 10;

Value = Undefined

Variables are often declared without a value. The value can be something that has to be calculated, or something that will be provided later, like user input.

Re-Declaring Variables

Re-declaraing a JavaScript varaibles will not lose it's value.

Arithmetic

You can do arithmetic with JavaScript variables by using the operators like = and *;
Example: var i = 1 + 2 + 3;
