Comments

Used to explain code, and make it more understandable and more readable. It also used to prevent execution when testing alternative code.

Single Line Comments

This starts with double slash (//). Any text between // and end of the line will not be executed by JavaScript.
Example:
//Change Heading
//Change Paragraph

Multi-line Comments

This starts with /* and ends with */ signs.
Example:
/*
The code below is etc. etc.
*/

Preventing Execution

Adding double slash (//) in front of the code line will change the code lines from executable lines to a comment.
//document.getElementById("JavaSEg").innerHTML = "First Page";
document.getElementById("JScriptEg") = "First Paragraph";
