Introduction

JavaScript is easy to learn. It is the Programming Language of the Web. It can update both HTML and CSS. Data manipulation, validation, and calculation are the common uses of JavaScript.

Changing HTML content

The method getElementById() is use to "find" HTML element and converting the element content (innerHTML).
Example: document.getElementById("example").innerHTML = "JScript";

Change HTML Styles (CSS)

JavaScript can change the style of an HTML element.
Example: document.getElementById("example").style.fontSize = "20px";

Hide HTML Elements

JavaScript can hide elements (HTML elements) by using the display style:
Example: document.getElementById("example").style.display = "none";

Show HTML Elements

JavaScript can also show hidden elements (HTML elements) by using the display style:
Example: document.getElementById("example").style.display = "block";
