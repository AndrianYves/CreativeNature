Arithmetic

Arithmetic Operations usually operates on two numbers.
Example: var x = 100 + 40;
or variables: var x = a + z;


or expressions: var x = (100 + 40) / q;

Operators and Operands

The arithmetic operation numbers are called operands.

The operator to be performed between two operands is defined by operator.

  • Operand = 100; Operator = +; Operand = 50

Adding

Plus sign (+) or the addition operator is used for adding numbers.
Example: var x = 5;
var i = 1;
var q = x + i;

Subtracting

Minus sign (-) or the subtraction operator is used for subtracting numbers.
Example: var x = 5;
var i = 1;
var q = x - i;

Multiplying

Multiplication sign (*) or the multiplication operator is used for multiplying numbers.
Example: var x = 5;
var i = 1;
var q = x * i;

Dividing

Division sign (/) or the division operator is used for dividing numbers.
Example: var x = 5;
var i = 1;
var q = x / i;

Remainder

Modulus operator (%) returns the division remainder.
Example: var x = 5;
var i = 2;
var q = x % i;

Incrementing

Increment Operator (++) is used for incrementing numbers.
Example: var x = 5;
x++;
var i = x

Decrementing

Decrement operator (--) is used for decrementing numbers.
Example: var x = 5;
x--;
var i = x

Exponentiation

Exponentiation operator (**) will raise the first operand to the power of the second operand.
Example: var x = 1;
var z = x ** 2;

Operatir Precedence

This describes the order in which operations are performed in an arithmetic expression.
