Dates

Creating Date Objects

Dates are created with the new Date() constructor.

4 ways to create a new date objects:

  • new Date()
  • new Date(year, month, day, hours, minutes, seconds, milliseconds)
  • new Date(milliseconds)
  • new Date(date string)

new Date()

This method creates a new date object with the current date and time.
Example: var det = new Date();

new Date(year, month, ...)

This method creates a new date object for a specific date and time.
Example: var det = new Date(2019, 11, 24, 10, 33, 30, 0);

Previous Century

1 and 2 digits years will be described as 19xx.
Example: var det = new Date(99, 11, 24);

new Date(dateString)

This method creates a new date ibject from a date string.
Example: var det = new Date("March 30, 2019 11:13:00");

new Date(milliseconds)

This method creates a new date object as zero time plus milliseconds.
Example: var det = new Date(0);
