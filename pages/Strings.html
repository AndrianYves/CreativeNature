Strings

Used to store and manipulate text. String is zero or more characters wirtten inside quotes.

  • var i = "John"; //Using Double Quotes
  • var i = 'John'; //Using Single Quotes

Backslash escape character

Backslash escape character (\) turns special characters into string characters.

  • Code (\') = Result (') = Single Quote
  • Code (\") = Result (") = Double Quote
  • Code (\\) = Result (\) = Backslash

Other escape sequences

  • \b = Backspace
  • \f = Form Feed
  • \n = New Line
  • \r = Carriage Return
  • \t = Horizontal Tabulator
  • \v = Vertical Tabulator
