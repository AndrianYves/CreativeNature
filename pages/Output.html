Display Possibilities

JavaScript can also display data in other ways:

  • innerHTML, for writing into an HTML element
  • document.write(), for writing into the HTML output
  • window.alert(), for writing into an alert box
  • console.log(), for writing into the browser code

Using innerHTML()

Example:

document.getElementById("example").innerHTML = 9 + 1;

Using document.write

Example:

document.write(9 + 1);

Using window.alert()

Example:

window.alert(9 + 1);

Using console.log()

Example:

console.log(9 + 1);
