Display Possibilities

JavaScript can also display data in other ways:

  • innerHTML, for writing into an HTML element
  • document.write(), for writing into the HTML output
  • window.alert(), for writing into an alert box
  • console.log(), for writing into the browser code

Using innerHTML()

Use document.getElementById(id) to access an HTML element.

Example:

document.getElementById("example").innerHTML = 9 + 1;

Using document.write

Use document.write method for tesing purposes.

Example:

document.write(9 + 1);

Using window.alert()

Use window.alert() (alert box) method to display data.

Example:

window.alert(9 + 1);

Using console.log()

Use console.log() method for debugging purposes.

Exmaple:

console.log(9 + 1);
