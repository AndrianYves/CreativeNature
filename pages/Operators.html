Operators

Assignment Operators (=) are used to assign values to a variable.
Example: var i = 2;

\

Addition Operator (+) are used to add numbers.
Example:
var z = 10;
var i = 10;
var y = z + i;

Multiplication Operator (*) used to multiply numbers
Example:
var a = 1;
var s = 2;
var d = a + s;

Arithmetic Operators

  • + = Addition
  • - = Subtraction
  • * = Multiplication
  • ** = Exponentiatin
  • / = Division
  • % = Modulus
  • ++ = Increment
  • -- = Decrement

Assignment Operators

  • "=" = x = y or y = x
  • += = x += y or x = x + y
  • -= = x -= y or x = x - y
  • *= = x *= y or x = x * y
  • /= = x /= y or x = x / y
  • %= = x %= y or x = x % y
  • **= x **= y or x = x ** y

String Operators

Plus sign (+) operator is used to add strings.
Example:
var p1 = "Hi";
var p2 = "Hello";
var p3 = p1 + p2;

String and Numbers

Adding a number and a string will return a string.
Example:
var a = "Hi" + 5; = Hi5

Logical Operators

  • && = logical and
  • || = logical or
  • ! = logical not

Type Operators

  • typeof = Returns the type of a variable
  • instanceof = Returns true if an object is an instanve of an object type.

Bitwise Operators

  • & = AND
  • | = OR
  • ~ = NOT
  • ^ = XOR
  • << = Zero fill left shift
  • >> = Signed right shift
  • >>> = Zerp fill right shift
