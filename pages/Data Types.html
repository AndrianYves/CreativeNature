Data Types

JavaScript can hold many data types: Objects, Strings, Numbers, etc.
Example:
var width = 20;
var firstName = "John";
var i = {country: "Philippines", city:"Baguio"}

Strings

Text Strings is a series of characters. It is written using single or double quotes.
Example:
var name = "John"; //Using double quotes
var name = 'John'; //Using single quotes

Numbers

Numbers can be written with or without decimal.
Example:
var i = 20; //With decimal
var i = 20.00; //Without decimal

Booleans

Booleans have two values: true or false.
var a = 1;
var b = 1;
var c = 3;
(a == b) //Returns True
(a == c) //Returns False

Arrays

Written with square brackets. Typically separated by commas.
Example: var cars = =["Honda", "Peugeot", "Ford"];

Objects

Written using curly braces.

Written as name:value pairs, then separated by a comma.
Example: var individual = {lastName:"Doncic", firstName"Luka"};

typeof Operator

The typeof Operator use to find the type of a JavaScript varaible. It returns the type of a varaible or an expression.
Example:
typeof "" //Returns string
typeof "John" //Returns string

Undefined

A variable with a value has the value of undefined.
Example: var car; //Value is undefined

Values

Empty Strings has both valid value and a type.
Example: var name = "";

Null

Null suppoed to be something that doesn't actually exist, but in JavaScript, null is an object.

Primitive Data

A single simple data value with no added properties or methods

The typeof operator can actually return one of the following primitive types:

  • number
  • boolean
  • string
  • undefined

Complex Data

typeof Operator can return one of the two complex types:

  • object
  • funtion

typeof Operator returns object for arrays, null, and objects. It doesn't return object for any functions.
