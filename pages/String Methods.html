String Methods

String methods and properties are also available to primitive values.

Length

length property returns the length of the string.
Example:
var eg = "ASDFGH";
var eg1 = eg.length;

Finding a String in a String

indexOf()

Using indexOf() method will return the index of the first occurrence of the specific text.
Example:
var eg = "Locate this";
var find = eg.indexOf("this");

lastIndexOf()

Using lastIndexOf() method will return the index of the first occurrence of the specific text.
Example:
var eg = "Locate this 'this' ";
var find = eg.lastIndexOf("this");

Searching for a String in a String

search() method used for searching a string in the specific value.
Example:
var eg = "Locate this";
var find = eg.search("this");

Extracting String Parts

3 methods for extracting a part of a string:

  • substring(start, end)
  • substr(start, length)
  • slice(start, end)

slice() Method

This method extracts a part of string. It returns the extracted part that results to a new string.
Example:
var eg = "Apple, Banana, Kiwi";
var res = eg.slice(7, 13);

subtring() Method

This method is similar to slice(), but this method doesn't accept negative indexes.
Example:
var eg = "Apple, Banana, Kiwi";
var res = eg.substring(7, 13);

substr Method

This method is similar to slice(), the difference is the second parameter will have the specific length of the extracted part.
Example:
var eg = "Apple, Banana, Kiwi";
var res = eg.substr(7, 6);

Replacing String Content

replace() Method will replace the specific value with another value of a string.
Example:
eg = "Please visit my house";
var res = eg.replace("my", "your");

Convert Upper and Lower Case

toUpperCase() method is use to convert a string to upper case. While toLowerCase() method is use to convert a string to lower case.
Example:
var eg = "Luh";
var res = eg.toUpperCase(); //Upper Case

var eg = "Luh";
var res = eg.toLowerCase(); //Upper Case

concat() Method

concat() Method joins two or more strings.
Example:
var eg = "Luh";
var eg1 = "Luh1";
var res = eg.concat(" ", eg1); //Upper Case

trim()

This method is use to remove whitespaces from strings.
var eg = " Luh ";
alert(eg.tring());

Extracting String Characters

3 methods for extracting string characters:

  • charCodeAt(position)
  • chartAt(position)
  • Property access []

chartAt() Method

This method returns the character of a specific index in a string.
var eg = "Luh";
eg.chartAt(0);

chartCodeAt() Method

This method will return the unicode of the character of a specific index in a string.
var eg = "Luh";
eg.charCodeAt(0); //Upper Case

Property Access

This method allows property access [] on strings.
var eg = "Luh";
eg[0]; //Upper Case
