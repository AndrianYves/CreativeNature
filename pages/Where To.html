Where To

The codes of JavaScript should be inserted between starting script tag and end script tag:
Example:
script
document.getElementById("example").innerHTML = "JScript";
/script

Functions and Events

JavaScript function block can be executed when "called" for.

Javascript in Head tag or Body tag

The user can place any amount of scripts in HTML document. It can be placed in the Body or Head part of HTML page.

Java Script in Head Tag


Example:

head
script
function myFunction(){
document.getElementById("example").innerHTML = "Jscript Change";
}
/script
/head

JavaScript in Body Tag


Example:

body
script
function myFunction(){
document.getElementById("example").innerHTML = "JScript Change";
} /script
/body

External JavaScript

JavaScript uses .js as file extension. To use external JavaScript, insert the name of the script int src attribute of the beginning script tag.


Example:
(script src="theScript.js")(/script)

External References

External Scripts can be also referenced by using URL or folder.


Example:
For URL: (script src="http://www.facebook.com")(/script)
For Folder: (script src="/js/theScript.js")(/script)
