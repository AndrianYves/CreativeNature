Arrays

Arrays is used to store multiple values in a single variable.
Example: var name = ["John", "Ralph"]

Definition

It is a special variable, that holds more than one value at a time.

Creating an Array

The easiest way to create an Array is by using array literal.
Syntax: var name_array = [item, item1, ...];

JavaScript Keyword new

Example of creating an Array and assigning values to it:
var name = new Array("John", "Ralph");

Elements of an Array

Referring th index number can acces an array element.
Example: var name = cars[0];

Changing an Array Element

Changes the value of the first element.
Example: name[0] = "John";

Access the Full Array

Referring the array name can access the full array.
Example: var name = ["John", "Ralph"];
document.getElementById("Example").innetHtml = name;

Arrays are Object

typeOf operator in JScript will return an object for arrays.
Example: var name = ["John", "Ralph", "20"];

Array Elements Can Be Objects

JScript variables can be objects.
Example: myArray[0] = Date.now;

Array Properties and Methods

Arrays are the built-in array properties and methods.
Example: var x = name.length;

Length Property

length property of an array will return the length of an array.
Example: var name = ["John", "Ralph"];
name.length;

Looping Array Elements

Using for loop is the safest way to loop through an array.

Adding Array Elements

Using th push() method is the easiest way to add a new element to an array.
