Number Methods

Number methods and properties are also available to primitive values.

toString() Method

This method will return a number as a string.
var z = 123;
z.toString();
(123).toString();
(100 + 23).toString();

toExponential() Method

This method will return a string with a number rounded and written using exponential notation.
var x = 9.656;
x.toExponential(2);
x.toExponential(4);
x.toExponential(6);

toFixed() Method

This method will return a string with the number written with a specified number of decimals.
var x = 9.656;
x.toFixed(0);
x.toFixed(2);
x.toFixed(4);
x.toFixed(6);

toPrecision() Method

This method returns a string with a numnber written for a specific length.
var x = 9.656;
x.toPrecision();
x.toPrecision(2);
x.toPrecision(4);
x.toPrecision(6);

valueOf() Method

valueOf() method will return a number as a method.
var x = 123;
x.valueOf();
(123).valueOf();
