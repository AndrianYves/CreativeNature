Semicolons

Semicolons usually separate JavaScript statements
Add a semicolon at the end of each statement to execute the statement.

Example:

var x, i, z;
x = 6;
i = 3;
z = x + i;

White Space

Add white space to the script to make it more understandalbe and readable.
Example:

With white Space: var person = "Yves";
Without white space: var person="Yves";

Line Length and Line Breaks

In case if a statement doesn't fit on a single line, place it after an operator:
Example:

document.getElementById("example").innerHTML =
"Hello Word!";

Code Blocks

Statements can also be merged together in code blocks, inside the curly brackets {...}.
It is used to execute JavaScript Statements together.
Example:

function myFunction() {
document.getElementById("example1").innerHTML = "Hello World!";
document.getElementById("example2").innerHTML = "Anya Ngay?";
}

Keywords

  • break - Terminates a loop
  • continue - Starts at the top after getting out of the loop
  • debugger - Calls the debugging function after stoppping the exectuion of JavaScript
  • do ... while - While a condition is true, it executes block of JavaScript statements and repeat the block
  • for - As long as the condition is true, it marks a block of JavaScript statements to execute.
  • function - Daclares functions
  • if ... else - It marks a block of JavaScript Statements that will be executed according to the condition.
  • return - Exits function
  • switch - It marks a block of JavaScript Statements that will be executed accroding to different cases.
  • try ... catch - Executes error handling to a block of JavaScript Statements.
  • var - Declares Variable
