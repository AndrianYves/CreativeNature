Topic 11 - Text

Text formatting with the use of CSS is fun.
It gives your content some boost in design.
With text formatting, you can also place your text wherever you wanted.

Example Code for Text Formatting 


h1{

    color: red;
    text-decoration: underline;
    text-indent: 50px;

}

p{
    color:black;
    text-decoration: overline;
    letter-spacing: -3px;
    word-spacing: 10px;

}

This are just few ways on how you can design your text.
