Topic 5 - Backgrounds

Just like the last topic, in here you can also set a background for your website using CSS.
It can either be an image or just a touch of your favorite color.


Example Code for Colors

body{
      background-color: black;
}

Example Code for Images

body{
      background-image: url("image.gif");
}

You can also set on how your background would appear.

Example Code

body{
       background-color: black;
       background-repeat: no-repeat;
        background-position: center;
}
