Topic 6 - Borders

In CSS you can specify the style, width or the color of your border.
There are various of styles to choose from for example
a dashed border style defined a dashed border.


Example Code for Borders

h1.dashed{
      border-style: dashed;
}

You can also set the width and height of your border.

Example Code for Borders

h1.dashed{
      border-style: dashed;
      border-width: 200px;
      border-height: 500px;
}

This are just few ways on how you can design your border.
