Topic 12 - Fonts

In this topic will go around about Fonts in CSS.
Just like the last topic, fonts also gives some boost in placing the right font in your content.
There are various of fonts available, ill cover some here.

Font-style is used to make your letters bold or italic.

Example Code for Text Formatting 


h1{

    font-family: "Times New Roman";
    font-style: normal;

}

p{
    color:black;
    font-style: italic;

}

This are just few ways on how you can design your fonts.
